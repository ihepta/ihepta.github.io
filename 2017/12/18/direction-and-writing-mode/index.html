direction

简介

该属性用来设置块级元素中的内容排布方向，可选属性值有：

ltr：从左到右，默认值

rtl：从右到左

inherit：继承自父元素

兼容性

chrome safari firefox opera ie
2.0+ 1.3+ any 9.2+ 5.5+

使用方法

对于内联元素块

首先了解一下何为“内联元素块”？

内联元素块：包括替换元素，如img、input、button、video、object等或者inline-block级别的元素。

针对行内元素，默认情况下，我们平时写html时，DOM在前的元素显示在左边，DOM在后的元素显示在右边，而如果设置了direction: rtl ，DOM在前的就会显示在右边，在后的就会显示在左边。如下所示：
